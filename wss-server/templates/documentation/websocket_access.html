Websocket

Haozheng Li
Create Date : 4 Mar, 2023
New

WSS platform supports access to devices with Websocket protocol. This article introduces the access process of using Websocket connection communication.

Restrictions

  • Only directly connected devices support communication using the Websocket protocol. Gateway devices and gateway sub-devices do not support communication using the Websocket protocol.
  • Websocket protocol access only supports Websocket request paths with TLS encryption. (e.g: wss://)
  • The size of data transmitted by the data uplink interface is limited to 128 KB.

Access process

The access process mainly includes: performing device authentication to obtain a device token, and using the obtained token for continuous data reporting.
